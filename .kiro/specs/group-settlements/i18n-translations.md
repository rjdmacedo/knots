### Merge instructions

1. Open `messages/{locale}.json`
2. Inside `"Balances"`, add `"Actions": { ... }` after `"description"` (or before `"Reimbursements"`)
3. Replace/update the existing `"Reimbursements"` object with the new keys (keep structure, update copy)
4. Preserve all other keys in the file
5. Run `node -e "JSON.parse(require('fs').readFileSync('messages/XX.json'))"` for each file to validate

### Usage in components

```typescript
const t = useTranslations('Balances.Actions')
const t_reimb = useTranslations('Balances.Reimbursements')
```
